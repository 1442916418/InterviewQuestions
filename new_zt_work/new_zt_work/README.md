## 正式环境版本
### v1.4.7

``` bash
# 安装 package.json 插件
yarn install

# 增加插件
yarn add <name>

# 删除插件
yarn remove <name>

# 开发环境本地运行
yarn run dev

# 生成环境打包
yarn run build

# 生产环境构建并查看bundle analyzer报告
yarn run build --report
```

## 注意  
依赖包请使用正在使用的依赖包文件夹，不要使用 yarn 安装
